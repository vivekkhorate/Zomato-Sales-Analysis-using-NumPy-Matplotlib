# 🍽️ Zomato Sales Analysis using NumPy

## 📌 Project Overview

This project performs **restaurant sales analysis similar to Zomato data** using **Python and NumPy**.
The goal of the project is to analyze yearly restaurant sales and understand **sales trends, performance, and cumulative growth**.

It demonstrates basic **data analysis techniques used in Data Analyst roles**, including numerical computation and visualization.

---

## 🎯 Project Objectives

* Analyze restaurant sales data across multiple years
* Calculate total yearly sales
* Identify minimum and maximum restaurant sales
* Compute cumulative sales growth
* Visualize sales trends using charts

---

## 🛠️ Technologies Used

* **Python**
* **NumPy**
* **Matplotlib**
* **Jupyter Notebook**

---



## 🔍 Analysis Performed

### 1️⃣ Total Sales Per Year

Calculate total sales across all restaurants.

### 2️⃣ Minimum Sales by Restaurant

Identify restaurants with the lowest sales performance.

### 3️⃣ Cumulative Sales Growth

Compute cumulative sales over years to analyze growth.

### 4️⃣ Sales Trend Visualization

Plot the average cumulative sales using **Matplotlib line charts**.
